# Bosta-Task
## Getting started 
* Clone the repository 
```bash
git clone https://github.com/omar-bakr/Bosta-Task.git
``` 
* Install dependencies

```bash
npm install 
```

* Create an env file in the root directory simialr to .env-example file and add your env variables

* Start the server 

```bash
npm start
```
